# Persona Tags

给 SillyTavern 的 User 人设加标签的插件。

人设一多就乱，分不清谁是谁，所以写了这个。打个标签，筛选一下，就清楚了。

## 有什么用

- **打标签**：右栏给人设加标签，支持逗号分隔一次加多个
- **按标签筛选**：左栏下拉选标签，只显示匹配的人设
- **正向/反向筛选**：正向是只看选中的，反向是把选中的藏起来
- **标签排序**：筛选面板里长按标签可以拖动排序，顺序会记住
- **按绑定筛选**：默认只看当前角色卡绑定的人设，也能切到全部
- **绑定标签高亮**：当前角色卡绑定的人设带的标签，筛选面板里会微微发亮
- **人设卡片上显示标签**：每个人设下面直接能看到它的标签
- **选人设弹窗增强**：多个人设绑同一个角色卡时，弹出来的选择框能看到头像、名字、标签
- **复制人设**：复制时可以勾选是否带标签、是否带绑定关系，勾选状态会记住
- **弹窗汉化**：复制确认、人设选择等英文弹窗都换成了中文

## 安装

把 `persona-tags` 文件夹放到：

```
SillyTavern/public/scripts/extensions/third-party/persona-tags/
```

刷新页面就能用。

## 数据

标签数据存在 SillyTavern 的 `extension_settings` 里，不碰原有数据。
